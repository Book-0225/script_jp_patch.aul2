# AviUtl2 スクリプト日本語化パッチ
AviUtl2のスクリプトに含まれる英語パラメータ・項目名を日本語化し、可読性を向上させる言語ファイル（`.aul2`）です。

## ダウンロード・インストール
https://github.com/Garech-mas/script_jp_patch.aul2/releases/latest

上記のリンクから、環境に合わせていずれか1つをダウンロードしてください。
- 標準版: `script_jp_patch.au2pkg.zip`
- アクセスキー付き日本語リソース版: `jp_akey_aul2_script_jp_patch.au2pkg.zip`<br>
※ [アクセスキー付き日本語リソース](https://github.com/Garech-mas/jp_akey.aul2) を利用している方向けです。

AviUtl2を起動し、ダウンロードしたZIPファイルをプレビュー画面にドラッグ＆ドロップすることで導入できます。

再起動後、メニューバー **設定** > **言語の設定** から、`スクリプト日本語化パッチ` を選択してください。

- 本ファイルはスクリプトの表示内容を変更するのみであり、元のスクリプトの挙動には影響しません。
